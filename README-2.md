## 🚀 How to Deploy on GitHub Pages

### Step 1 — Create a GitHub Repository

1. Go to [github.com](https://github.com) and log in
2. Click the **"+"** button → **"New repository"**
3. Name it `ergroup-website` (or any name you prefer)
4. Set it to **Public**
5. Click **"Create repository"**

### Step 2 — Upload the Files

1. Inside your new repository, click **"Add file"** → **"Upload files"**
2. Drag and drop both `index.html` and `README.md`
3. Scroll down and click **"Commit changes"**

### Step 3 — Enable GitHub Pages

1. Go to your repository **Settings** (top menu)
2. Scroll down to **"Pages"** in the left sidebar
3. Under **"Source"**, select **Branch: `main`** and folder: **`/ (root)`**
4. Click **Save**

✅ Your website will be live in 1–2 minutes at:
```
https://yourusername.github.io/ergroup-website
```
